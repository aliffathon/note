Hugo Nofancy

Quick Start

Static Content Generator with HuGO

Install Latest GO

Download Site:

https://golang.org/dl/

download versi terbaru untuk OS mu, misal go1.15.linux-amd64.tar.gz untuk Linux Mint 18.3 64bit

Cara Installnya, cukup extract ke /usr/local utk dipakai secara global di sistem atau di $HOME/.bin untuk dipakai di user tersebut saja,

cd ~
cd Downloads
sudo tar -C /usr/local -xzf go1.15.linux-amd64.tar.gz   #global
tar -C $HOME/.bin -xzf go1.15.linux-amd64.tar.gz        #local

kemudian masukkan path hasil extract tadi ke environment variable sistem

# global
nano /etc/profile
export PATH=$PATH:/usr/local/go/bin

# local
nano $HOME/.profile #local
export PATH=$PATH:$HOME/.bin/go/bin

Close & Buka Terminal Baru, cek versi GO

go version
go version go1.15 linux/amd64

Uninstall GO

# global
sudo rm -rf /usr/local/go

# local
rm -rf $HOME/.bin/go

hapus path GO /usr/bin/go/bin dari /etc/profile atau $HOME/.bin/go/bin dari $HOME/.profile

Install HuGO

Secara lengkap bisa dilihat disini

Membuat site baru

cd
hugo new site hello     # site kosong/default
cd hello
hugo new theme empty    # theme kosong/default

Menambahkan Theme, perlukah theme? theme adalah template cara content di-generate, berisi layout dan assets. Cara memakai theme cukup tambahkan di config.toml atau Copas isi theme ke dalam layout dll

cd hello
git init
git submodule add https://github.com/gizak/nofancy themes/nofancy
echo 'theme = "nofancy"' >> config.toml

Jalankan hugo

hugo server
hugo server -t nofancy -D

keterangan:

  • -t --theme theme yg dipakai
  • -D generate draft post/content

Meng-generate pada directory tertentu

hugo -d /tmp/hello

Membuat Content dg theme nofancy

hugo new posts/nofancy.md

Header Content, menjelaskan bagaimana content di proses, diletakkan paling atas, sebelum content.

title: "Hugo Nofancy"
date: 2020-08-19T11:46:19+07:00
type: post
tags: 
 - hugo
categories:
 - hugo
draft: false

save dan lihat di http://localhost:1313/
